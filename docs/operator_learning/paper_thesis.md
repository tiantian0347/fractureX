## 记号与连续问题(应力-位移混合式)

域 Ω⊂ℝ²,损伤 d∈[0,1],退化函数 g(d)=(1−d)²(+k 残余刚度)。
弹性以 Hellinger–Reissner 混合(应力为原始变量)写:令 A(d)=(g(d)+k)⁻¹C⁻¹ 为
退化柔度,Σ=H(div,Ω;𝕊)(对称张量、法向连续),V=L²(Ω;ℝ²)。求 (σ,u)∈Σ×V:

  (A(d)σ, τ) + (div τ, u) = ⟨u_D, τn⟩_{Γ_D},  ∀τ∈Σ,  τn=g_N(=0) on Γ_N
  (div σ, v) = −(f, v),                          ∀v∈V

- **Γ_D(位移边界)= 自然边界**,经 RHS ⟨u_D,τn⟩ 弱施加;u_D=0 处贡献为 0。
- **Γ_N(面力边界)= 本质边界**,对 σ 施加 τn=g_N。
- model0:顶边 u_y=load、内圆 u=0 ∈ Γ_D;自由边 traction-free ∈ Γ_N。
  (推论:离散时**位移边界必须正确归入 Γ_D**;若误入 Γ_N 被强加 σn=0,则该边
  成"自由孔"→ 欠约束。这正是数据 wrapper 曾经的 bug,见 [[surrogate_data_underresolved_hl0]]。)

注:平衡 `div σ + f = 0` 对 σ 恒成立,**与 d 无关**;退化只进柔度 A(d)。

## P1 — 为什么 Hu-Zhang 应力是"正确的监督目标"

Hu-Zhang 元:Σ_h⊂H(div,𝕊),次数 k;V_h 为间断 P_{k−1}。三条性质作地基:

1. **一致 inf-sup 稳定** ⇒ 离散混合问题适定。
2. **离散平衡(关键)**:div Σ_h ⊆ V_h,故第二式给出
   **div σ_h = −Π_{V_h} f 逐元精确成立**(H(div) 协调混合法的标志)。
   即 σ_h **局部/逐元平衡**,而非仅弱平衡。
3. **σ 的最优先验估计**:‖σ−σ_h‖_{H(div)} ≤ C h^k|σ|_{k+1},**不依赖位移、无 locking**。

**对照**:标准位移有限元 σ_h^rec=C ε(u_h) 靠微分恢复 ⇒ σ_h^rec∉H(div)(法向不连续)、
**不满足离散平衡**、阶低且含噪。⇒ 监督源质量本质不同。

## 中心命题(平衡缺陷下界 —— 论文的硬锤)

设监督目标为 σ\*,代理 σ̂ 在 H(div) 意义逼近 σ\*。则平衡残差满足

  | R(σ̂) − R(σ\*) | ≤ ‖div(σ̂ − σ\*)‖ .

**推论(必要性)**:R(σ̂) → R(σ\*) 当 σ̂→σ\* 于 H(div)。即**代理能达到的最好平衡 = 监督目标的平衡**:
- σ\*=σ_h(Hu-Zhang):R(σ_h)=‖f−Π_{V_h}f‖=O(h^k) 最优小 ⇒ 代理**可**近平衡;
- σ\*=σ_h^rec(位移 FEM 恢复):R(σ_h^rec)=δ 不随 h 最优消失 ⇒ **代理平衡残差被 δ 下界锁死**。

> **结论:想要"平衡的代理",必须用"平衡的数据"监督。** Hu-Zhang 由此从"更好"升级为"必要"。
> 这是别人(用位移-FEM σ 的工作)结构上拿不到的判别度。

# 附:三个严格化

记 𝒯_h 为 Ω 的 shape-regular 单纯形网格,Hu-Zhang 应力空间 Σ_h⊂H(div,Ω;𝕊)(次数 k,
本工作 k=3),位移空间 V_h = 间断 P_{k−1}(本工作 P_2);Π_h:L²(Ω;ℝ²)→V_h 为 L² 正交投影;
体力 f∈L²(Ω;ℝ²)(断裂算例通常 f≡0)。

## A. 离散平衡(P1 的精确措辞 + 引文)

**结构性质(引文级,非本文证)**:Hu-Zhang 对 (Σ_h,V_h) 在 shape-regular 网格上满足

  **div Σ_h = V_h**(满射)  与  一致离散 inf-sup。

> 措辞建议:"The Hu–Zhang pair (Σ_h,V_h) of degree k satisfies div Σ_h = V_h together with a
> uniform discrete inf-sup condition." 引:Hu & Zhang(对称应力混合元族,2D 三角形 / nD 单纯形)
> 与 Arnold–Falk–Winther 的 FEEC 恰当列(exact sequence / commuting diagram)框架。
> [按你用的具体元核对确切文献年份]

**引理 A(离散平衡恒等式)**:设混合法离散方程 `(div σ_h, v_h) = −(f, v_h) ∀v_h∈V_h` 成立。
若 `div Σ_h ⊆ V_h`,则在 L²(Ω) 中

  **div σ_h = −Π_h f**,   等价地  **div σ_h + f = f − Π_h f ⟂ V_h**。

特别地 **f∈V_h(如 f≡0)⇒ div σ_h = −f 逐点精确**(f≡0 时 σ_h 精确无散)。

*证*:令 w_h := div σ_h + Π_h f。因 div σ_h∈V_h 且 Π_h f∈V_h,有 w_h∈V_h。对任意 v_h∈V_h,
`(w_h,v_h) = (div σ_h+f,v_h) − (f−Π_h f,v_h) = 0 − 0 = 0`(前项是离散方程,后项由 L² 投影
正交性 f−Π_h f⟂V_h)。取 v_h=w_h 得 ‖w_h‖_{L²}=0。∎

> **平衡的两层含义(务必区分)**:本文所谓"平衡精确"指**离散混合方程意义**——`div σ_h=−Π_h f`
> 在 V_h 上**精确成立**(f∈V_h 时逐点精确)。它**逼近**而非等同于**连续 PDE 的逐点平衡** `div σ+f=0`:
> 二者相差**载荷投影/离散一致性误差** `f−Π_h f=O(h^k)`(f∈V_h 时为 0)。下文凡称 σ_h"平衡/无散",
> 均指此离散意义;连续逐点平衡按离散一致性以 O(h^k) 逼近。

## B. 中心命题(平衡缺陷下界):陈述 + 假设 + 证明

定义**平衡残差泛函**:对 τ∈H(div,Ω;𝕊),

  R(τ) := ‖ div τ + f ‖_{L²(Ω)} .

**命题 B(平衡继承 / 缺陷下界)**
设监督目标 σ\*∈H(div,Ω;𝕊) 与代理 σ̂∈H(div,Ω;𝕊),且

  (H) ‖div(σ̂ − σ\*)‖_{L²(Ω)} ≤ ε .

则

  **R(σ\*) − ε ≤ R(σ̂) ≤ R(σ\*) + ε** ,  即 |R(σ̂) − R(σ\*)| ≤ ε .

*证*:三角不等式。R(σ̂)=‖div σ̂+f‖=‖(div σ\*+f)+div(σ̂−σ\*)‖≤‖div σ\*+f‖+‖div(σ̂−σ\*)‖
= R(σ\*)+ε;同理 R(σ̂)≥R(σ\*)−ε。∎

**推论 B1(Hu-Zhang 监督可达近平衡)**:取 σ\*=σ_h。由引理 A,R(σ_h)=‖f−Π_h f‖≤C h^k|f|_{H^k}
(若 f∈V_h 则 =0)。故 R(σ̂) ≤ ‖f−Π_h f‖ + ε(f≡0 时 ≤ ε)。

**推论 B2(非平衡监督锁死下界)**:若监督源在所选残差范数下有不可消平衡缺陷 R(σ\*)=δ>0
(典型:位移有限元微分恢复应力 σ_h^rec,法向跨边不连续、不属 H(div),其
broken-div + 法向跳量缺陷 δ>0),则 **R(σ̂) ≥ δ − ε**。即 ε<δ 的代理也不可能比 δ−ε 更平衡——
**监督目标的平衡缺陷给代理设了硬下界**。⇒ 想要平衡代理,必须用平衡数据(Hu-Zhang)监督。

> 抽象推广(覆盖 σ_h^rec∉H(div) 情形):令 B:X→Y 有界线性,R(τ)=‖Bτ+f‖_Y。则 R 关于
> 半范 |τ|=‖Bτ‖_Y 是 1-Lipschitz,命题对 (B=div, X=H(div)) 与 (B=broken-div⊕法向跳量) 同时成立。

**关键 Remark(诚实点 → Stage D 损失的动机)**:训练通常控制 **L²** 误差 ‖σ̂−σ\*‖_{L²},
而假设 (H) 要的是 **散度半范** ‖div(σ̂−σ\*)‖。二者不等价:散度涉及导数,
`‖div_h(σ̂−σ\*)‖ ≲ h^{−1}‖σ̂−σ\*‖_{L²}`(差分放大 1/h)。故**仅 L² 监督不保证 (H)**。
推论:要兑现平衡保证,需在损失里**显式加平衡项** λ‖D_h σ̂ + f‖²(H(div)-aware / 物理信息监督)
—— 这正是 **Stage D 的损失**。而无论怎么训,命题 B 的下界 R(σ̂)≥R(σ\*)−ε 都在,
故 Hu-Zhang(R(σ\*)≈0)仍是必要前提。

## C. 网格平衡残差 R_h(σ̂)(精确离散定义 = Stage D 指标)

代理在 Cartesian 网格(H×W,行 i↔y、列 j↔x,步长 Δx,Δy)上输出
σ̂=(σ̂_xx,σ̂_yy,σ̂_xy),掩码 m=Ω 指示。平衡 div σ + f = 0 的分量:
∂_xσ_xx+∂_yσ_xy+f_x=0、∂_xσ_xy+∂_yσ_yy+f_y=0。

**有效域** Ω_h^∘ := {(i,j): m=1 且 中心差分四邻点全在 Ω 内};另**排除全损伤格**(d>d_c)以免
裂纹处差分噪声主导。中心差分:

  r_x[i,j] = (σ_xx[i,j+1]−σ_xx[i,j−1])/(2Δx) + (σ_xy[i+1,j]−σ_xy[i−1,j])/(2Δy) + f_x
  r_y[i,j] = (σ_xy[i,j+1]−σ_xy[i,j−1])/(2Δx) + (σ_yy[i+1,j]−σ_yy[i−1,j])/(2Δy) + f_y

**绝对残差**(L²(Ω) 一致求积):R_h(σ̂) = ( Σ_{Ω_h^∘}(r_x²+r_y²) ΔxΔy )^{1/2}。

**无量纲/尺度无关版(跨样本可比,= Stage D 报告量)**:

  R̃_h(σ̂) = L · ‖(r_x,r_y)‖_{L²(Ω_h^∘)} / ‖σ̂‖_{L²(Ω_h^∘)} ,  L = diam(Ω) .

(σ 已按 stress_scale 归一,故 R̃_h 中归一因子抵消;量纲:div σ ~ 应力/长度,×L/应力 → O(1)。)

> **定位**:R_h / R̃_h 是**可实现的离散近似指标**(有限差分 + 掩码 + 求积),用于**训练正则与评估报告**;
> 它**不等同于连续残差** `‖div σ+f‖` 本身,而是以差分一致性 O(h_g^p) 逼近之(并在裂纹带受噪声放大,
> 故限定 Ω_h^∘)。命题 B/B1 中的连续量 R 与此工程指标 R_h 经一致性相联系,二者勿混。
**一致性**:对 σ_h 在网格上采样,smooth 区 R_h(σ_h)→R(σ_h)(差分一致);裂纹带差分放大噪声,
故限定 Ω_h^∘ 排除 d>d_c 并同时报"全域 vs 起裂前区"两版(见 [[phasefield_sanity_check]] 诚实标注)。

**实现接口(替换 metrics.py 的 NotImplementedError)**:
`equilibrium_residual_l2(sigma_grid, mask, dx, dy, f=0.0, d=None, d_c=0.9) -> float`,
纯 numpy,返回 R̃_h;Stage D 同名损失项用 R_h(可微版,torch)。

## D. T1:亚网格过程区峰**在本文数据管线与模型类下不可稳定恢复**(三道分辨率闸 + 回归均值)

**目的**:把 §P3 "逐点裂尖 σ 峰不可学"从经验上升为定理,并解释为何幅值比对**损失/导出分辨率
鲁棒**。诚实地说,这**不是**单一"网络表示下界",而是流水线上**三道串联闸** + 一个**回归均值**机制;
最强之处是它**反推出**我们已观察到的三件事(下文 D.5)。

> **适用边界(务必声明)**:本下界针对**本文的数据生成(FE 网格 h_FE)、导出分辨率(h_grid)与模型类**
> (网格采样的谱型/卷积型算子,及 L² 风险下的最优预测)。它**不主张对所有算子学习架构普适成立**——
> 例如奇异富集(预测 K + 解析型)、tip-局部坐标、无网格/图算子等可绕过相应闸(见后文构造性出口)。
> 即:T1 是"在本文流水线与模型类下,逐点峰被三闸+回归均值锁低"的**条件性**下界,非普适不可能性定理。

### D.0 过程区峰的模型
AT2 1D 最优损伤剖面 d(x)=e^{−|x|/(2l0)},σ=g(d)σ₀。把裂尖前沿应力建模为**光滑背景 + 局部凸起**:
σ(x;a) = σ_smooth(x;a) + b_a(x),  b_a(x) = A(a)·φ((x−ξ(a))/l0),
φ 为固定单位剖面(∫φ²=1、∫φ≠0),A(a)=峰幅、ξ(a)=裂尖位置,a=样本参数。
注 ‖b_a‖²_{L²}=A² l0‖φ²‖、L∞ 峰 = A·‖φ‖_∞。判据用归一化幅值比
η := ‖σ̂‖_{peak}/‖σ‖_{peak}(论文里报的就是它,见 §6.2、§8)。

### D.1 闸一 — 采样/投影闸(FE 网格 & 导出网格,数据侧)
**引理 T1a**:设 Π_h 为网格尺度 h 上的 L² 投影(或节点插值)到分片 P_m。则凸起的**可表示峰**满足

  ‖Π_h b_a‖_{∞} ≲ C_m · (l0/h) · A,   故  η ≲ C_m·(l0/h)  当 h ≳ l0.

*证(略要)*:含凸起的单元宽 h ≫ l0,b_a 的质量集中在 ~l0 内;P_m 在该元上的最优拟合峰受单元均值
控制,(1/h)∫b_a = A(l0/h)∫φ。高阶 P_m 捕获更多但同阶 O(l0/h)→0。∎
⇒ **只要 FE 网格或导出网格中较粗者 h ≳ l0,峰在该处即被压扁**,与训练无关。

### D.2 闸二 — 限带闸(谱型算子,FNO 理想化)
**引理 T1b**:若 σ̂ 限带到 K 模(理想 FNO,谱卷积截断)。b_a 的谱 b̂_a(k)=A l0 φ̂(l0 k)e^{−ikξ},
尺度 1/l0 衰减。若 K < L/(2π l0),

  ‖b_a − P_K b_a‖²_{L²}/‖b_a‖²_{L²} ≥ c(φ, K l0/L) > 0  (Parseval,截断尾能量).

**诚实警示**:FNO 的逐点非线性会展宽频谱,故 T1b 严格只对**线性谱模型类**;对真实 FNO 是
该类的下界(理想化)。对 UNet 取**有限感受野/分片常值**版:格内常值无法表示亚格变化,退回 T1a 型采样下界。

### D.3 闸三 — 回归均值闸(L² 风险 + 裂尖位置不确定,训练侧、最深)
**命题 T1c**:设输入仅把裂尖位置 ξ 决定到分辨 Δ(粗网格混叠 ⇒ ξ|inputs 散布宽 ~Δ)。
L² 风险下 Bayes 最优预测 = 条件均值 σ̄(x)=E[b_a(x)|inputs]=A·E_ξ[φ((x−ξ)/l0)] = b ∗ ρ_ξ(卷积平滑)。
则**峰被系统性低估**:

  η\* := peak(σ̄)/A ≈ ( 1 + c·(Δ/l0)² )^{−1/2} < 1.

⇒ **即便无穷容量 + 数据已解析峰**,只要输入不能精确定位裂尖(Δ≳l0),最优算子也**系统性欠预测**
峰幅 —— 解释了为何观测到的是**偏低**(η<1),而非随机误差。

### D.4 流水线推论(主结果)
峰相对误差/幅值比受**最弱闸**支配:

  η ≲ min{ C_m·l0/max(h_FE, h_grid),  限带项,  η\*(Δ/l0) }.

特别地:**若 FE 网格欠解析(h_FE ≳ l0),峰在源头即失,与导出网格、网络、损失无关**;
修复数据(h_FE ≤ l0/2)后,绑定闸转为导出网格、限带与回归均值。

### D.5 与实验的自洽(T1 的最强证据:它反推已知现象)
- **幅值比~0.5**(§6.2、§8):欠解析数据 h_FE/l0≈5 ⇒ 闸一 η≲C_m·l0/h_FE ~ 0.2–0.5(P2 捕获更多),量级吻合。

  换损失不改 ⇒ 调损失推不动峰。✓

## E. D1:解算子在起裂处不连续(算子学习对断裂是否良定义)

**为何必答**:神经算子的万能逼近(FNO:Kovachki 等;DeepONet:Chen–Chen、Lanthaler 等)要求被逼近
算子在紧集上**连续/Lipschitz**。而准静态断裂的"参数/载荷 → 解"映射在成核处**跳变**。不正面处理,
审稿人会质疑整套学习是否良定义。下面给出精确刻画 + 诚实出路。

### E.0 解算子
固定参数 a=(几何, Gc, l0, E, ν) 与单调载荷 {t_n}。增量(不可逆)极小:
(u_n,d_n)∈argmin E_{t_n}(u,d) s.t. d≥d_{n−1}。待学算子 G:(a,{t_n}) ↦ {(d_n,σ_n)};
或 G_a:t ↦ (d(t),σ(t))。

### E.1 不连续性(精确陈述)
把"成核 → 不可逼近"拆成**两条独立命题**(避免综合论证):一条只讲**场映射不连续**,一条只讲
**网格收敛泛函可学**;"连续算子不可逼近"作为命题 E1 的直接推论。

**命题 E1(场映射不连续)**:E_t 非凸,载荷控制全局极小下存在临界载荷 t_c(a),使

  lim_{t↑t_c}(d_t,σ_t) ≠ lim_{t↓t_c}(d_t,σ_t)   于 L²(Ω),

即 t↦(d,σ) 在 t_c **跳变**(一阶相变 / snap-back / 对称破缺分岔),不连续集 Σ_disc={t=t_c(a)}。
> 引:Francfort–Marigo 全局极小的 BV-in-time 轨迹与跳;Mielke–Roubíček 率无关系统的跳解。

**推论 E1′(连续算子不可一致逼近)**:神经算子 Ĝ 连续(各层连续之复合)。连续函数跨越幅为 J 的阶跃
在过渡层误差 ≥ ½J ⇒ 在 Σ_disc 任意正测度邻域 ‖Ĝ−G‖_{L²}≥c·J>0。**场级演化在成核处不可一致逼近。**

### E.2 命题 E2(网格收敛泛函可学)
**命题 E2**:下列泛函**对参数 a 连续**(故在连续算子类内可逼近):
   - **峰值载荷** F_max(a)=t_c(a)·(起裂前刚度):t_c(a) 为两分支能量交点载荷,随 a 连续(除测度零高阶退化);
   - **总耗散能** / 能量-载荷曲线值函数(min 值函数连续,仅在 t_c 有拐点,非跳);
   - **最终裂纹路径**:除测度零分岔集外随 a 连续。
**推论 E2′(连续演化下场可学)**:若改用连续演化(粘性正则 ḋ 惩罚 / 弧长延拓穿过 snap-back),
t↦state 换成弧长 s↦state 为 **Lipschitz** ⇒ 重参数化后**场轨迹可学**。
> ⇒ 论文只对命题 E2 的网格收敛泛函声称精度(与 §P3 一致);场级预测要么避开 Σ_disc,要么走 E2′ 连续演化数据。
> 引:Knees–Mielke / Negri 的粘性 → 参数化连续解。

### E.3 数值签名 = 起裂处 staggered 爆炸(与你项目统一)
**注 E3**:t_c 处正是交错不动点失去压缩(更新映射谱半径→1/>1)之处——这就是相场起裂迭代发散的根。
**理论不连续与数值不收敛是同一现象**。故:数据在 Σ_disc 邻域不可信(交错残差不下降),应在该处
用连续延拓(弧长 / 粘性 / Anderson 加速)或显式标注剔除。呼应 [[staggered_acceleration_refs]]、[[phasefield_sanity_check]]。

### E.4 D1 与 T1c 同币两面
成核处裂尖**初生**,其位置对 a 最敏感(恰在不连续集上)⇒ T1c 的裂尖定位不确定 Δ 在此最大。
**最难区(成核)= G 跳变处(D1)= 裂尖定位最差处(T1c)= 数值最不收敛处(E3)**。四者同源。
⇒ 统一信息:**代理对"网格收敛泛函"又快又准且良定义;对"成核瞬时场"既不连续又定位差又数值脆——
列为不可学习残余难量**,与 P3/T1 闭环。

## F. T2:恢复应力的平衡缺陷下界(命题 B2 的定量牙齿)

**目的**:把命题 B2 里"非平衡监督的缺陷 δ>0"做成**显式、residual-estimator 风格**的下界,并钉住
**定性区别**:HZ 监督的缺陷**可被训练约化到 ~ε(reducible)**,恢复应力的缺陷是**与网络/训练/数据量
无关的不可约下界(irreducible floor)**。

### F.0 恢复应力
位移有限元(连续 P_m,degree m)解 u_h,**恢复应力** σ_h^rec = g(d_h) C ε(u_h):分片 P_{m−1}、对称,
但**法向跨边不连续** ⇒ σ_h^rec ∉ H(div)。其分布散度含边上 Dirac(跳量),故朴素 ‖div σ_h^rec+f‖_{L²}
不适用——须用对偶/估计子范数。

### F.1 平衡缺陷的正确度量
定义弱平衡缺陷(H^{−1} 残差,等价于 residual estimator):

  δ(τ) := ‖f + div τ‖_{H^{−1}(Ω)} = sup_{v∈H¹₀,‖v‖₁=1} [ (f,v) − (τ, ε(v)) ] .

逐元分部 ⇒ δ(τ) 由**单元残差** r_K=div τ+f 与**法向跳量** J_e=[[τ·n]] 共同决定;经典 residual estimator

  η(τ)² := Σ_K h_K²‖r_K‖²_{L²(K)} + Σ_e h_e‖J_e‖²_{L²(e)}

满足两侧等价 **c₁ η ≤ δ ≤ c₂ η**(可靠性 δ≤c₂η + 有效性 c₁η≤δ),c₁,c₂ 仅依赖形状正则性与多项式次数,
**与 h 无关**。引:Verfürth(残差估计子可靠+有效);Braess–Schöberl(equilibrated flux);Ainsworth–Oden。
> **本节下界明确建立在此两侧等价之上**:Θ(h^m) 的下界来自**有效性方向** c₁η≤δ(估计子下界 ⇒ δ 下界),
> 而非仅可靠性;故需估计子的标准假设(形状正则、光滑精确解、振荡项 osc(f)=h.o.t.)。

### F.2 引理 T2(下界 + 与 HZ 对比)
设 σ_h^rec 如上,精确应力 σ 在所取局部光滑(远离裂尖,见 P3);估计子两侧等价之标准假设成立。则
**(a) 不可约跳量下界**(依赖**有效性** c₁η≤δ):δ(σ_h^rec) ≥ c₁ (Σ_e h_e‖J_e‖²_e)^{1/2}。除非 σ_h^rec∈H(div)
  (即精确应力被连续通量空间复现——非平凡反例外不成立),**跳量项严格 >0**;再由两侧等价
  δ≃‖σ−σ_h^rec‖_{L²},对光滑 σ 得 **δ=Θ(h^m)**(上界由可靠性、下界由有效性,双向夹紧)。
**(b) HZ 对照**:Hu-Zhang σ_h **法向连续 ⇒ J_e≡0(所有边)**,且 div σ_h=−Π_h f ⇒
  δ(σ_h)=‖f−Π_h f‖_{H^{−1}}=O(h^{k+1})(f∈V_h(如 f≡0)时 **=0**)。
⇒ **δ(σ_h^rec)=Θ(h^m) 由跳量主导且不可去;δ(σ_h)=0 或 O(h^{k+1}) 且无跳量**。

### F.3 对命题 B2 的定量化(核心)
代入命题 B(平衡继承):R(σ̂) ≥ δ(σ\*) − ε。
- σ\*=σ_h^rec:**R(σ̂) ≥ Θ(h^m) − ε**。该 Θ(h^m) 是**数据侧固有 floor**:与网络容量、训练损失、
  样本量**无关**(再多数据、再调损失、加 Stage D 项也压不破,因为目标本身就带这缺陷)。
- σ\*=σ_h:R(σ̂) ≤ δ(σ_h)+ε = ε(f≡0)。**缺陷只剩训练误差 ε,可约**(更多数据/Stage D 项→0)。

> **定性区别(论文该强调的)**:不是"δ_rec 数值大",而是 **reducible vs irreducible** ——
> HZ 监督的平衡残差受训练支配(可降),恢复应力监督的受**数据结构下界锁死**(不可降)。
> 这是结构性、非调参可破的区别,正是 Hu-Zhang"必要而非更好"的定量证据。

### F.4 诚实备注 + 对照实验靶子
- **equilibrated flux recovery**(Braess–Schöberl / Ladevèze)能把位移解后处理成 H(div) 平衡应力——
  但那是**额外机器**,且本质在**重建 Hu-Zhang 原生就有的性质**;HZ 把平衡应力作为**原始变量**直接、
  最优、无后处理给出。论文据此立"原生平衡 vs 后处理/恢复"。
- **对照实验(坐实 F.3)**:同架构同数据量,分别用 σ_h(HZ)与 σ_h^rec(位移FEM)监督,报 R̃_h(σ̂):
  预期 HZ 组 →ε(随 Stage D/数据量降),rec 组**停在 ~δ_rec=Θ(h^m) 平台**(降不动)。这张"平台 vs 下降"
  曲线就是命题 B2 + T2 的实验证据。

## G. T3:L² 监督不保证平衡;Stage D 项的必要性与一致性

**目的**:把 §B Remark / §F.3 埋的"L²≠平衡"形式化成命题,证明 **Stage D 平衡损失项必要**,并第二次
推出 **Hu-Zhang 必要**(平衡项仅在平衡数据上与拟合一致;在恢复应力上二者冲突)。

记 L² 监督 E(σ̂)=‖σ̂−σ\*‖_{L²},平衡残差 R(σ̂)=‖div σ̂+f‖(网格上 R_h=‖D_hσ̂+f‖)。

### G.1 命题 T3a(L² 不控平衡,且 1/h 放大尖锐)
> **框架声明**:本命题的**可操作陈述是 (ii) 网格 + 数值导数框架**(R_h 用有限差分 D_h);(i) 连续版
> 仅作机理说明,需 σ̂ 在 H(div) 外/导数无界处取扰动,在本文的网格离散设定下由 (ii) 的 1/h_g 放大具体化。
> 即:本节不依赖连续函数空间的一般性,只断言**在网格离散与数值导数下** L² 监督不控 R_h。

**(i) 连续(机理说明)**:∀ε,M>0,∃σ̂ 使 ‖σ̂−σ\*‖_{L²}<ε 但 ‖div(σ̂−σ\*)‖>M。
  *证*:取 σ̂=σ\*+δ_n,δ_n=(1/√n)ψ(n·)·E₀(ψ 光滑 O(1) 振荡,E₀ 固定方向)。
  则 ‖δ_n‖_{L²}=O(n^{−1/2})→0,而 ‖div δ_n‖=O(n^{1/2})→∞。∎
  ⇒ **L² 接近与平衡可任意脱钩**(散度是导数,低幅高频扰动即破)。
**(ii) 网格 + 数值导数(可操作的主陈述,两侧 + 尖锐)**:在尺度 h_g 网格、中心差分 D_h 下,
  R_h(σ̂) ≤ R_h(σ\*)+‖D_h‖·E(σ̂),其中 **‖D_h‖=Θ(h_g^{−1})**(差分算子谱范,Nyquist 频率达到)。
  故把 E 训到 ε 只保证 R_h(σ̂) ≤ R_h(σ\*)+C·ε/h_g —— **平衡证书随 h_g→0 退化**,且 1/h_g 放大尖锐
  (Nyquist 网格扰动达到)。**这是本节的严格主张**(限定于上述网格离散+数值导数设定)。

### G.2 命题 T3b(Stage D 项:必要性、控制、一致性)
Stage D 复合损失 L_D(σ̂)=E(σ̂)² + λ R_h(σ̂)²。
**(a) 必要性**:由 T3a,**任何只含 E 的损失都不能给出平衡证书**;显式 R_h 项(λ>0)是平衡保证之必需。
**(b) 控制**:R_h(σ̂)² ≤ L_D(σ̂)/λ ⇒ 训到 L_D≤τ 即 R_h≤√(τ/λ);并由命题 B,
  R(σ̂) ≤ R(σ\*)+√(τ/λ) 被**证书化**。
**(c) 一致性 ⇒ 又一次 Hu-Zhang 必要**:
  - σ\*=σ_h(HZ,f≡0):div σ\*=0 ⇒ R_h(σ\*)=O(h_g^p)(仅 FD 截断)。故 **L_D 与 E 的极小在 σ\* 处相容**——
    平衡项**不偏置**拟合,只加一个可忽略一致项。
  - σ\*=σ_h^rec:R(σ\*)=δ_rec=Θ(h^m)>0(§F)。则 **min E(→σ\*) 与 min R_h(→0) 冲突**:无法同时 σ̂=σ\*
    且平衡 ⇒ Stage D 在拟合与平衡间**权衡**,最优解既不拟合数据也不平衡。
  ⇒ **平衡训练只在平衡数据上自洽**;恢复应力上 Stage D 反而暴露其内在矛盾。

### G.3 定理链综述(论文理论主干)
把 B + T2 + T3 串起来,即"平衡保持代理"的**结构最优配方**(措辞见下方收紧版):

  R(σ̂) ≥ R(σ\*) − ε   (命题 B,下界)
  R(σ\*) = 0 (Hu-Zhang, f≡0)  vs  Θ(h^m) (恢复应力,§F/T2)   (数据侧)
  L² 监督仅控 R 到 O(ε/h_g);需 Stage D 项 λR_h² 方证书化(§G/T3)   (训练侧)
  Stage D 项仅在 R(σ\*)=0 的数据上与拟合一致(§G.2c)   (再证 HZ 必要)

> **结论(可作为论文主定理的陈述,收紧版)**:*在本文设定与数值假设下,Hu-Zhang 平衡监督配合平衡
> 正则,是实现低平衡残差代理的**结构最优方案**:辅以平衡正则的算子代理可达平衡残差 R(σ̂)=O(ε)
> (随训练/数据可约);而任何基于非平衡(微分恢复)应力的监督,其平衡残差被数据固有缺陷 Θ(h^m)
> 不可约地下界锁死,且平衡正则与数据拟合相冲突。* 这是本文的理论立身点,P3/T1/D1 划出其诚实边界
> (逐点成核场不可学)。
> （不主张"必要且充分"这一最强表述:充分性是结构性的,必要性以本文的监督类与残差度量为前提。）

## H. B1:J 积分误差定理(平衡残差 → 断裂驱动力 G/K 的误差界)

**目的**:把"裂尖逐点应力不可学(T1)、但驱动量 G/K 可经围道积分准确取出"做成**带误差界的定理**,
且误差由**中心指标 R(σ̂)** 控制 ⟹ 把主论点(平衡保持)与断裂量(G/K)定量焊死。

### H.0 J 积分(域形式)+ 适用前提
> **适用前提(务必声明)**:本定理用的是**经典 LEFM 的近似域形式 J 积分**,前提:
> ① 积分环 A 是**光滑环域**,取在**过程区之外、远离损伤核**(d≈0、材料近线弹性、σ 光滑)处;
> ② 该环内**无显著损伤梯度/体力/惯性**(否则需相场构形力/额外项修正);③ 裂面近似无面力。
> 相场断裂里 J 积分须谨慎:**只在上述光滑环域成立**,不主张穿过过程区。后文 G/K 误差界均在此前提下。

裂尖周围取**光滑环** A(过程区之外、精确解光滑),权 q(内边界=1、外边界=0)。无体力时能量释放率

  G = ∫_A ( σ_ij ∂u_i/∂x₁ − W δ_{1j} ) ∂q/∂x_j dA,  W=½σ:ε.

连续介质中**路径无关 ⟺ 平衡 div σ=0 + 相容 + 裂面无面力**(均在光滑环域内取)。

### H.1 定理 B1(J 积分误差界)
设 (σ̂,û) 为代理场、(σ,u) 为参考,环 A 在光滑区。记环上平衡残差 R_A(σ̂)=‖div σ̂+f‖_{L²(A)}。
代理能量释放率 Ĝ:=J[σ̂,û;q] 满足

  **|Ĝ − G| ≤ C₁ ‖(σ̂−σ, û−u)‖_{H¹(A)} + C₂ R_A(σ̂)** ,

其中 C₁=C₁(‖σ‖,‖σ̂‖,‖∇u‖,‖∇q‖),C₂=C₂(‖∂û/∂x₁‖_{L²(A)},‖q‖_∞)。

*证(略要)*:域形式恒等式给出
J[σ̂,û;q] = (G-形式,场的双线性项) + ∫_A (div σ̂+f)·(∂û/∂x₁) q dA − ∫_A f·(∂û/∂x₁) q dA。
对**精确**(σ,u),平衡 div σ+f=0 使中间项消失、回到 G。故
|Ĝ−G| ≤ |J[σ̂,û]−J[σ,u]|(场误差,环上 J 关于场 Lipschitz ⟹ ≤C₁·H¹ 误差) +
|∫_A (div σ̂+f)·∂û/∂x₁ q|(平衡残差项 ⟹ ≤C₂R_A)。∎

### H.2 推论(论文用)
- **平衡保持 ⟹ G/K 准且不受 T1 限制**:若 R_A(σ̂)→0(B2 构造 / Stage D + HZ 数据),则
  Ĝ→G 仅以**环上光滑场**的逼近率收敛——**绕开裂尖未解析峰(T1)**,因为 G/K 由光滑环决定。
  (K 由 G=K²/E′ 或相互作用积分分离模式,误差结构相同。)
- **路径无关性 = R 的 a posteriori 指示**:两围道差 |J[σ̂;Γ₁]−J[σ̂;Γ₂]| ≤ C R(σ̂)(围域内)。⟹
  **J 随围道的离散程度 = R 的可计算误差棒**,给 Ĝ 自带置信区间(并接 §B4 最优 q)。
- **第三个可学泛函**:G/K(经 J 积分)与峰值载荷、裂纹路径并列为**网格收敛、可学、可加速**的量;
  且其精度由本文中心量 R 控制。引:Rice 1968(J);Shih–Moran(域形式);相场 J/构形力变体。

## I. B2:结构保持算子(可选**增强实现方案**,非基础定理)

> **定位声明**:本节是一个**可选的增强实现方案**(architecture choice),**不属于主结论的基础定理**。
> 主线(A→B→T2→T3)与诚实边界(T1/D1)、J 积分(B1)**不依赖** B2 即成立。B2 仅说明:若把平衡从
> Stage D 软惩罚换成 Airy 势**硬约束**,可得 R≡0 的更强实现;它是"how to do better",不是"what must hold"。

**目的(本节范围内)**:给出平衡**按构造精确**(R≡0)的一种参数化,并指出它与 Hu-Zhang 数据的相容性
(从模型类一侧再次印证监督源应平衡)。

### I.0 Airy 势参数化(2D,f≡0)
对称应力满足 div σ=0 ⟺ 存在标量 Airy 势 φ 使 σ=𝒥(φ):=curl curl φ,显式

  σ_xx=∂²φ/∂y²,  σ_yy=∂²φ/∂x²,  σ_xy=−∂²φ/∂x∂y  ⟹  div σ≡0 (逐点恒等).

这是 **2D 弹性复形** `C^∞(Ω) --𝒥(Airy)--> C^∞(Ω;𝕊) --div--> C^∞(Ω;ℝ²)` 的恰当性(range 𝒥 = ker div);
与 Hu-Zhang(H(div) 端的协调离散)**同根**于 Arnold–Falk–Winther 弹性复形,只是从**势端**进入。

**构造**:算子预测光滑标量势 φ̂,取二阶导得 σ̂=𝒥(φ̂)(f≠0 时加固定特解 σ_p,div σ_p=−f)。

### I.1 命题 B2(平衡按构造 + 模型类必要性)
设 σ̂=𝒥(φ̂)(+σ_p)。则 **div σ̂+f=0 逐点成立 ⟹ R(σ̂)=0**(至微分离散截断)。因此:
**(i)** 由命题 B,R(σ̂)=0 **与训练/数据无关** ⟹ 代理**精确平衡保持**;
**(ii)** 由 B1,J 积分平衡残差项消失 ⟹ Ĝ 仅受环上场误差限;
**(iii)** Stage D 平衡软项**不再必要**,T3c 一致性冲突**消解**。
**(iv) 模型类一侧再推 Hu-Zhang 必要**:模型类 {𝒥(φ)} = **无散对称应力**。拟合**非平衡**目标 σ\*(恢复
应力,div σ\*≠0)**本质不相容**:min_φ‖𝒥(φ)−σ\*‖ 的不可约残差 = σ\* 的**非无散分量** = Θ(δ_rec)(§F)。
⟹ **无散模型拟合非平衡数据,E_σ 被 δ_rec 锁死**;而平衡数据 σ\*=σ_h(div=0)∈range 𝒥,模型类**能精确表示**
⟹ 无矛盾。**故结构保持算子与 Hu-Zhang 数据互为前提**(目标∈模型类),共同给出精确平衡+可拟合。

### I.2 诚实备注(trade-off)
- **相容性变成另一半约束**:σ=𝒥(φ) 自动平衡,但 ε=A(d)σ 须**相容**(Saint-Venant / Beltrami–Michell:
  curl curl(A σ)=0;均质各向同性 f≡0 时即 **Δ²φ=0** 双调和)。⟹ 若只自由预测 φ,平衡精确但相容近似。
  实务:**联合预测 (φ̂, û)** 并罚本构失配 ‖A(d)𝒥(φ̂)−ε(û)‖(把软残差从平衡换到**本构/相容**)。
  对主论点这是**正确的取舍**:平衡正是控制 J 路径无关(B1)的量,令其精确最划算。
- **二阶微分放大噪声**(同 §G 的 1/h):φ̂ 须低频/光滑预测;近裂尖 2 阶导放大与 **T1 正交**——
  **B2 修平衡、不修分辨率**;逐点峰仍需 §"奇异富集"。两者组合:B2(平衡)+ 富集(峰)。
- 引:Arnold–Falk–Winther(弹性复形 / FEEC,Airy 为其势);结构保持/物理约束算子学习。

### I.3 与定理链的接续
B2 把主定理的"训练侧"从"软罚 R(T3)"升级为"**硬约束 R≡0**":此时
R(σ̂)=0(构造) ⟹ 命题 B 给 R(σ̂)=0、B1 给 Ĝ 仅受光滑场误差限、且(I.1iv)Hu-Zhang 数据仍必要(拟合相容)。
⟹ **"结构保持算子(Airy)+ Hu-Zhang 平衡数据"= 平衡保持代理的最强(硬约束)实现**,
Stage D(软)为其退化/无势参数化时的替代。
